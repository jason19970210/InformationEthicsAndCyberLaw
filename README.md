智產權：

著作權法僅保護表達，不保護思想
著作完成時()，即取得著作權，不需經審查
四個要件：原創性、客觀表達性、創作性、藝術學術範圍、(非著作權保護之標的)
著作財產權：屬雇主
著作人格權：公開發表之權利、屬著作者
